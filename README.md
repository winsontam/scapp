## Getting Started
- Clone this repository.
- Install node dependencies.
```bash
npm install
```
- Install React Native CLI
```bash
npm install -g react-native-cli
```
- Start in iOS emulator
```bash
react-native run-ios
```

## Run in Xcode
- go to /ios
- open SCAPP.xcworkspace by Xcode
